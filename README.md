# PlacePicker 🗺️

A simple React app to build your personal collection of places you'd love to visit or already have visited.

## 🚀 Getting Started

1. Clone the repo
2. Run `npm install`
3. Run `npm start`

## 📸 Features

- Select places from a pre-defined list
- Remove selected places with confirmation
- Fully responsive UI
